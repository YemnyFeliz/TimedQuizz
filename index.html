Quick Coding Timed Quizz

You will be presented with questions related to coding.
Click the start button to take the quiz.
Once you click start, the timer will start and a question with multiple choices will appear.
When you select an answer you will be taken to the next question.
If the answer is wrong time will be taken off.
The quizz is over when the timer has reached 0 or when all questions have been answered.
